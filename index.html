About Me

Hey there! I'm Ryan, and I'm a distinct recent-graduate full-stack web and software developer based in Brisbane, Australia.

I love everything about tech and its incredible uses in the everyday world, having studied and graduated a Bachelor of Information Technology degree with distinction from the Queensland University of Technology with a double major in Computer Science and Data Science.

I specialise in creating amazing applications using a plethora of languages and tools, working within diverse and highly collaborative Agile teams backed with a keen attention to detail, creative problem-solving, passionate proactivity, and initiative.

Projects

Volcanoes of the World

A React-based web application for users to view and analyse data about volcanoes based and extending on a REST API for data retrieval and user authentication. Authorised and registered users gain access to extra information using data and authentication endpoints fetched and handled with HTTP GET and POST requests. The root serves Swagger documentation for API reference.

Technologies:

  • React.js
  • Node.js + Express.js
  • MySQL
  • Knex.js
  • JWT
  • Swagger

Raspberry Pi Music Diary Twitter Bot

An early project of mine and one I look back on fondly as a start to my coding journey. A music diary Twitter bot tweeting logs of what a user is or has been listening to accompanied by an image of said user, as well as a physical notifier of when a music-related tweet is detected on the X/Twitter platform. Makes use of the Twitter and Last.fm APIs. View a video demonstration of this project above!

Technologies:

  • Python
  • Raspberry Pi with an attached webcam and LED
  • tweepy library for Twitter API interactions
  • pylast library for Last.fm API interactions
  • picamera and opencv libraries for webcam image capture
